Introdução:

	Características de um app híbrido:

 • SPA - Single Page Application
   Aplicativo Web que contém somente uma página. Todo o conteúdo é
   carregado dinamicamente nesta página.
	• URL routes
	  Rotas URL apontam para caminhos virtuais que serão processados
      pelo JavaScript em vez de acessados pelo navegador.
 • PWA - Progressive Web Application
   Um site que passa a ser tratado como um aplicativo, lançando um
   modo especial dos navegadores modernos.
